# Security policy

WITShift is alpha software and should not be treated as a sandbox. Reports for security-sensitive
issues should include the affected version, minimal reproduction, impact, and whether untrusted input
is required. Do not include credentials, private source, or live exploit data in a public issue.

Use GitHub's private security-advisory channel for this repository when available. Otherwise contact
Antonio Antenore at `ant_ant95@hotmail.it` with the subject `WITShift security`.

Only the latest alpha release receives security fixes. Acknowledgement does not imply that a report
is accepted until it is reproduced and scoped.
